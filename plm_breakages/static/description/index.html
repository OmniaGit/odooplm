Breakage Management

This module allows you to handle breakage on product and BOM manufacture order

How it works:

  • A new menu is created in order to better manage your breakage



  • Once you click on it there is a graphical view of you breakage product



  • To create a new breakage just click on create and edit the form



  • On the BOM you have a button that shows you which element has breakage



  • On the Manufacturing order you have a button that shows you if you have breakage
